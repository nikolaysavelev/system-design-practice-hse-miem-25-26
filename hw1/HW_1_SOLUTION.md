### Какие метрики можно отслеживать

1. Технические метрики сетевых запросов (с разложением по эндпоинтам)
  - Количество запросов в секунду (RPS)
  - Доступность: количество успешных ответов к общему числу
  - Время обработки запроса - avg, min, max, разложение по перцентилям (50-95)

2. Метрики физических ресурсов системы:
  - CPU utilization
  - RAM utilization
  - RX/TX utilization
  - Disk utilization

3. Метрики базы данных:
  - количество активных соединений
  - количество транзакций в секунду
  - disk usage
  - среднее время жизни транзакции
